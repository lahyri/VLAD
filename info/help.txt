The Vampire Log Automated for Discord (A.K.A. VLAD) is a Vampire the Masquerade companion that aims to help your roleplay.

--- Commands ---
    -- !help            - list all the commands this bot has
    -- !r N D          - generates N random rolls of a d10 and compares them to the difficulty D, returning the result of the test
    -- !discipline D L  - Sends the description of the determinated D discipline in the L level. If called without parameters, returns a list with all the disciplines.

Carpe Noctem!
